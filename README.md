# TokTok

## How to Run

Navigate to the 'back' folder

```sh
cd back
```

Install node modules

```sh
npm install
```

Run the server

```sh
node server.js
```

## How to Update Front-End

Navigate to the 'front' folder

```sh
cd front
```

Install node modules

```sh
npm install
```

Build the React project for production

```sh
npm run build
```
